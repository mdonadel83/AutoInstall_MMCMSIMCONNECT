### What is this installer?

This installer automatically downloads and installs the **MMCM SimConnect** plugin for SimHub, always fetching the latest version directly from GitHub.

### Requirements

- **SimHub** installed on your PC
- **Internet connection** (to download files from GitHub)
- **Windows 10/11**

### How to install

1. **Download** both files and place them in the same folder:
   - `Installa_MMCM_SimConnect.bat`
   - `Install-MMCMSimConnect.ps1`

2. **Close SimHub** if it's running

3. **Double-click** on `Installa_MMCM_SimConnect.bat`

4. Follow the on-screen instructions:
   - The installer will automatically find your SimHub folder
   - It will ask for confirmation before proceeding
   - It will download and install the plugin files

5. **Launch SimHub** and go to the plugins menu to enable **MMCM SimConnect**

### Troubleshooting

| Problem | Solution |
|---------|----------|
| "SimHub not found" | Manually enter the path when prompted (e.g., `C:\Program Files\SimHub`) |
| "SimHub is running" | Completely close SimHub and try again |
| Connection error | Check your internet connection and try again |
| Windows blocks the script | Click "More info" and then "Run anyway" |

### Manual installation (alternative)

If you prefer to install manually:
1. Go to [GitHub Releases](https://github.com/mdonadel83/MMCMSimConnect/tree/main/Release/net48)
2. Download all files from the `Release/net48` folder
3. Copy them to your SimHub installation folder
